To rename a chat in Anti-Gravity IDE:
1. Change the name in the left sidebar list
2. Close the open tab (the open tab holds the old name in memory and will revert the sidebar if left open)
3. Double-click the renamed entry in the sidebar to reopen it
4. The tab now shows the new name and the rename sticks

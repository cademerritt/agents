Rule: Never write, edit, or create files—or take any system action—without explaining exactly what you're doing and getting explicit permission first. Each action must be approved individually.

**Why:** Cade has repeatedly said this, and past mistakes included silent deletions, GitHub pushes, and mid-task edits.

**How to apply:** Ask for permission before each file change; one "yes" only applies to that action.

**Exception:** During an abrupt shutdown, the F drive copy must happen—this drive is immutable, so even if the chat wasn't ended, the file must still be written there.
